# 家庭资产负债表 · 小程序

把信用扩张那套原理（资产 = 负债 + 净资产、杠杆放大、偿债覆盖）做成一个可自查的工具。

## 运行

1. 微信开发者工具 → **导入项目** → 选择本文件夹。
2. AppID 填 **测试号**（或保留 `touristappid` 用游客模式）。
3. 直接预览即可。已内置一份中等杠杆家庭的示例数据。

> 标题用思源宋体（Noto Serif SC），通过 `wx.loadFontFace` 远程加载。开发者工具里若要让它生效，勾选 **详情 → 本地设置 → 不校验合法域名**；加载失败会自动回退到系统衬线体，不影响使用。正式发布需在 mp 后台把字体域名加入 downloadFile 合法域名。

## 约定

- 资产、负债：**万元**。
- 月收入、月供：**元 / 月**。
- 数据只存在本机（`wx.setStorageSync`），不上传。

## 原理对应

| 界面 | 背后的式子 |
|---|---|
| 净资产双条 | 资产 = 负债 + 净资产（恒等式可视化） |
| 杠杆率（权益乘数） | 总资产 ÷ 净资产 |
| 资产负债率 | 总负债 ÷ 总资产 |
| 偿债收入比 | 月供 ÷ 月收入 |
| 应急储备 | 流动资产 ÷ 月供（能撑几个月） |
| 压力测试 | 净资产变动% = (房产现值 ÷ 净资产) × 房价变动% |
| 归零点 | 房价下跌 净资产÷房产现值 时，净资产清零 |

核心特色是压力测试那根"房价杠杆"：房贷固定，房价波动被 `房产现值 ÷ 净资产` 这个倍数放大进净资产——这正是上面讲的"涨 10% 净资产涨 30%、跌 33% 净资产归零"的双刃剑。
