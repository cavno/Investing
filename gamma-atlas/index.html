读图基础 — 四个核心元素

每张Gamma敞口图都由这四个元素构成。先建立"图样语言",再看14种结构才有意义。

零Gamma线
橙色竖线。正负Gamma分水岭 — 右侧波动被压制,左侧波动被放大。
现价线
灰色虚线。当前股价位置。相对零Gamma的位置,决定庄家行为模式。
看涨墙 (Call柱)
红色柱体。Call净Gamma暴露。柱子越高,该行权价的庄家对冲压力越大 — 形成阻力/磁吸。
看跌墙 (Put柱)
绿色柱体。Put净Gamma暴露。柱子越高,该行权价越像"地板/天花板" — 形成支撑或加速点。
总GEX曲线
蓝色平滑曲线。所有Gamma的累积叠加 — 显示在不同价位上,庄家整体对冲压力的"地形图"。

火箭点火型

Gamma Squeeze启动瞬间 — 价格刚突破上方密集Call墙

底层机制
  • 价格突破Call密集区,庄家对冲缺口暴增(Delta从0.3瞬间跳到0.8)
  • GEX曲线在突破点陡峭爬升 — 每涨1块都要追买几倍正股
  • 上方多个递增Call墙形成Gamma阶梯,逐级触发新一轮买入
触发条件

放量突破 + IV跳升 + Call大单流入,三者必须同时出现。

经典案例

2021年GME轧空、2024年NVDA多次突破关键整数关口、2023年特斯拉财报后跳涨。

深谷反弹型

负Gamma区Put墙托底 — 跌不下去只能向上释放

底层机制
  • 价格在负Gamma区,任何买盘都会被庄家追涨放大
  • 下方密集Put墙形成蹦床效应 — 跌不下去,能量只能向上释放

楼梯递进型

多级Call墙形成持续上攻 — 接力赛而非短跑

底层机制
  • 价格刚进入正Gamma区,上方是连续Call墙阶梯
  • 每攻破一级,庄家就要追加买入对冲下一级,形成接力式买盘
  • 与"火箭点火"的区别:火箭是单次爆发,楼梯是持续多日的稳定上涨
触发条件

第一级Call墙被放量突破 + 资金持续流入 + 不出现回吐。

经典案例

NVDA 2024年从500美元到950美元的多次阶梯式上涨、META 2023年单边复苏行情。

V型反转突破型

价格刚穿越零Gamma向上 — 政权交替的瞬间

底层机制
  • 价格刚刚从负Gamma区穿越到正Gamma区(从左到右)
  • GEX呈深V形,在Put积压区深陷负值,现价处快速回升至零线之上
  • 从"庄家追跌"模式切换到"庄家追涨"模式 — 这是政权交替的瞬间
触发条件

股价站稳零Gamma上方 + Put成交量萎缩 + IV回落但价格上行。零Gamma点本身在过去几小时逐步下移是关键标志。

经典案例

每次大跌后的"地量地价反弹"启动、2023年10月SPY双底反转。

零Gamma狙击型

临界点蓄势上膛 — 压到极致的弹簧

底层机制
  • 现价几乎正好坐在零Gamma点上(分水岭)
  • 关键不对称:上方Call墙明显强于下方Put → 上行阻力反成"磁吸",下行无明显支撑
  • 这是最不稳定的平衡态,任何小催化剂都会决定方向,但结构倾向于向上释放
触发条件

看催化剂方向 — 利好新闻、大盘普涨、板块共振都可能成为引信。一旦动起来速度极快。

风险提示

方向不确定,但发动后速度极快。需要快速跟进或退出,犹豫即出局。

№ 06 / 14 ↑ BULLISH · 周级

看跌墙瓦解型

Put大规模平仓引发庄家空头回补 — 最隐蔽的看涨信号

底层机制
  • 某些行权价的Put持仓量(OI)在快速下降,而股价没有大幅下跌
  • 大量Put被平仓 → 庄家原有的Put对冲(做空正股)失效
  • 庄家必须回补空头头寸(买入正股) → 形成被动买盘
关键观察

这是最隐蔽的看涨信号,在K线上几乎看不出来,只在期权数据中显现。看Put OI变化率(单日下降>10%)、Put/Call比从高位快速回落、IV Skew从陡峭转向平坦。

经典案例

财报公布利空但股价不跌时,常常是大空头被Put瓦解抛弃引发轧空。

悬崖跌破型

跌破Put墙触发负反馈 — Gamma Crash启动

底层机制
  • 价格刚刚跌破强Put墙,庄家原本买正股对冲Put的头寸全部失效
  • 庄家反手抛售正股 → 价格继续下跌 → 更多Put变实值 → 庄家继续抛
  • 这是典型的正反馈下跌(Gamma Crash),比正常下跌猛烈数倍
关键警惕

被跌破的Put墙反向变成阻力位,反弹困难。一旦启动几乎不可逆。

经典案例

2020年3月新冠崩盘、2022年Q2科技股崩盘的多个关键节点。

塔式雪崩型

负Gamma + 下方Put塔积压 — 多米诺骨牌效应

底层机制
  • 价格已经处于负Gamma区,庄家本已在追跌卖出
  • 下方多级Put墙等待被"点名" — 跌破后会触发新一轮卖出
  • GEX蓝线长期处于负值 → 整个价格区间都是"加速带"
触发条件

一旦下跌启动,会形成多米诺骨牌效应,逐级雪崩。

双顶失守型

强Call墙两次拒绝后回落 — M顶失败

底层机制
  • 现价在零Gamma上方,但距离强Call墙仍有距离且呈下行
  • Call墙特别厚(单根柱子异常突出),代表庄家有压倒性防守资源
  • 多空在Call墙周围反复肉搏,但攻方弹尽粮绝
  • Pinning效应反向 — 庄家从"压制"转为"驱赶下移"
触发条件

两次或多次接近Call墙都被快速打回(每次成交量递减)+ 跌破短期均线 + 大盘同步走弱。

经典案例

2022年7月SPY反弹到400附近被多次拒绝后的二次探底。

零Gamma穿越下行型

价格刚跌入负Gamma区 — V型反转的镜像

底层机制
  • 现价刚刚跌穿零Gamma点,从压制区进入放大区
  • 与V型反转突破型完全相反 — 庄家从"高抛低吸"切换为"追涨杀跌"
  • 零Gamma点本身在近期上移(说明Call变得不再有效,Put占主导)
风险特征

这种结构下,所有下方价位都是"加速带",反弹困难且容易失败。

经典案例

2022年9月SPY跌破4000整数关口、2024年8月日本"黑色星期一"的连锁触发。

虚空漂移型

上有铁顶下无支撑 — 折磨多头的阴跌

底层机制
  • 上方有明显的Call大墙(天花板),但下方没有像样的Put墙
  • 上方Call墙压制反弹,下方没有Put墙托底 → 庄家没有理由买入正股
  • 净效应:只有向下的力,没有向上的力,价格自然漂移走低
隐蔽性警告

这种结构不会暴跌,但会慢慢阴跌,持续一周到一个月。这是最折磨多头的结构 — 没有暴跌警报,但仓位天天缩水。

经典案例

2022年Q2-Q3的科技股阴跌期(没有崩盘日,但累计跌幅惊人)。

Pinning破裂下行型

平衡结构突然失守 — 横盘陷阱的终结

底层机制
  • 原本是平衡的双墙Pinning结构,但价格跌破下方Put墙
  • 原Pinning中的Put墙被击穿 → 庄家失去对冲基础
  • 上方Call墙瞬间"失效"(变得无关紧要)
  • 下方还有连串Put墙等待"被点名" → 每跌一级触发新一轮抛售
触发条件

Pinning持续多日后突然出现放量 + 单日跌幅显著超过Pinning区间宽度 + 跌破时无反弹。

经典案例

2023年3月SVB事件前,银行股长期Pinning突然崩盘。

双墙Pinning型

上下对称的区间震荡 — Pinning的标准教科书

结构特征
  • 上下两堵墙大致等大,价格被夹在中间
  • 庄家在两侧都有强对冲需求,任何方向移动都会被快速反向操作
  • 横盘震荡,日内波动小,成交量低迷
交易策略

适合卖出跨式期权(Short Strangle / Iron Condor) — 收时间价值。但要注意Pinning终会失效,失效方向需结合其他维度判断。

持续时间

可以持续数天到数周,直到一侧墙被消耗或新催化剂出现。

№ 14 / 14 ⇆ NEUTRAL · 高随机

无结构漂移型

任何结构都不显著 — Gamma分析的盲区

结构特征
  • 没有明显的Call墙或Put墙,所有行权价的Gamma都很小且均匀
  • GEX曲线非常平直(接近横线,起伏小),零Gamma位置不清晰
  • 庄家对冲压力小 → 股价几乎不受Gamma约束
常见原因
  • 该股票期权流动性差,机构参与少
  • 临近重大事件(财报、大选),所有人都在观望
  • 距离最近到期日时间较远,Gamma尚未集中
交易含义

Gamma分析在这种结构下基本失效,需要切换到其他分析框架(技术面、资金流、催化剂)。

§ 02 · METHODOLOGY

六层分析框架 — 结构只是其中一层

Gamma敞口是L2结构层。单独看它是片面的。完整的分析需要把六个层次叠加在一起,寻找共振。

L0
催化剂层
EVENT · NEWS · BLACK SWAN
财报、FOMC、CPI、地缘冲突、突发新闻。决定方向的根本动力,Gamma只放大它。
L1
情绪层
IV · PUT/CALL · SKEW
隐含波动率、Put/Call比、IV Skew。市场对未来不确定性的实时定价,最敏感的预警指标。
L2
结构层
GAMMA · OI WALLS · GEX
本图谱所讲的内容。庄家对冲压力的地形图,告诉你"如果...就会..."的条件性约束。
L3
资金层
FLOW · DARK POOL · BIG ORDER
实时资金流向、大单、暗池交易。告诉你真实意图 — 谁在动,动多猛,往哪流。
L4
价格层
PRICE · VOLUME · TECHNICAL
K线、成交量、技术形态。最表层的结果,但也是最直接的确认信号。
L5
仓位层
POSITION · RISK · KELLY
你的持仓与风控。模型再准也敌不过黑天鹅,这一层决定你能活多久。
